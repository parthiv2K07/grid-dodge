## Game Description
Grid Dodge is a browser-based game where the player avoids moving enemies inside a grid. 
The goal is to survive as long as possible and score points.

## How to Play
- Use Arrow Keys to move
- Avoid enemies
- Score increases with time
- Collision ends the game

## Key Features
- Start, Play, Game Over states
- Scoring system
- Keyboard controls
- Replay option

## Concepts Used
- Variables & Data Types
- Loops & Nested Loops
- Functions
- Arrays & Objects
- DOM Manipulation
- Event Handling
